# EWBF Exporter 

[Prometheus](https://prometheus.io/) exporter for the EWBF miner stats.

## Use

```sh
docker run -p 9100:9100 -it asbjornenge/ewbf-exporter:1.0.0 --miner_host 192.168.2.3 --miner_port 42000
```

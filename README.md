# EWBF Exporter 

Exporter for the EWBF miner stats to [prometheus](https://prometheus.io/) metrics.

## Use

```sh
docker run -it asbjornenge/ewbf-exporter --miner_host 192.168.2.3 --miner_port 42000
```
